# Disable Fatigue mod for Wurm Unlimited (Server)

Requires [Ago's Server Mod Launcher](https://github.com/ago1024/WurmServerModLauncher/releases) to run.

Fatigue doesn't make sense, when you play alone anyway. This mod removes fatigue from the game.
